Implementation of CCSDS hyperspectral image compression algorithm on FPGA

A sample adaptive linear compression algorithm was implemented using FPGA using a zynq-7000 SoC. Algorithm designed by Consusltative commitee for space data sciences. Completely pipelined datapath was developed for 3U nanosatellite with power and time optimisation.

User Space Library for LED P10 panels

Raspberry Pi was used to create a User-Space Library for cheap LED panels. FreeType was used for text rendering. Supports creation of large displays by assembling long daisy chains of individual panels. Existing SPI hardware on Raspberry Pi was used to implement HUB-12 protocol used by the panels.

Calibration of Ultrasound based flow meter DAU.

Ultrasonic Flow Meter was developed by Efftronics Systems for Urban Municipal corporation to monitor for leaks in the water grid. DAUs to be connected via GSM into and IoT framework. Undertook Calibration of Sensor modules to meet specific requirements. Work involved debugging existing embedded firmware.
